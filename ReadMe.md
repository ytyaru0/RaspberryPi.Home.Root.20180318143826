# このソフトウェアについて

日常スクリプト。やってみる体系の実装。

日々実装予定。RaspberryPi3 の`$HOME/root/`配下に配置。

# 概要

アウトプット自動化スクリプト。.shファイルを実行することで動く。一部の複雑な処理は.shから.pyを呼びだす形で実装。

# 使い方

1. ターミナルを起動する
1. `.bashrc`が呼び出される
1. 環境変数PATHに追加
1. `$ SS`, `$ push`などのコマンドが使用できる
